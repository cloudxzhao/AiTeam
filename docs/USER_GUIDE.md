## 1. 系统介绍

### 1.1 关于 AiTeam

AiTeam 是一款现代化的项目管理系统，专为敏捷开发团队设计。它提供了完整的项目管理工具链，帮助团队高效协作、追踪进度、管理任务。

### 1.2 核心功能

| 功能模块 | 说明 |
|----------|------|
| **项目管理** | 创建和管理多个项目，支持项目成员管理 |
| **看板系统** | 可视化任务管理，支持拖拽操作 |
| **用户故事** | 敏捷开发故事管理，支持故事点估算 |
| **问题跟踪** | Bug 和问题管理，支持优先级和严重程度 |
| **权限管理** | 灵活的角色权限体系，支持申请审批流程 |

### 1.3 系统要求

- **浏览器**: Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **网络**: 稳定的网络连接
- **分辨率**: 建议 1280x720 及以上

## 2. 快速入门

### 2.1 注册账号

1. 访问系统首页，点击「注册」按钮
2. 填写注册信息：
   - 用户名（必填，用于登录）
   - 邮箱地址（必填）
   - 密码（必填，至少8位）
   - 全名（选填）
3. 点击「注册」完成账号创建
4. 使用注册的用户名和密码登录系统

### 2.2 登录系统

1. 访问系统首页
2. 输入用户名和密码
3. 点击「登录」按钮
4. 登录成功后进入仪表盘页面

### 2.3 创建第一个项目

1. 点击左侧导航栏的「项目」
2. 点击「新建项目」按钮
3. 填写项目信息：
   - **项目名称**: 输入项目名称
   - **项目描述**: 描述项目内容（选填）
   - **可见性**: 选择公开或私有
4. 点击「创建」完成项目创建

### 2.4 邀请团队成员

1. 进入项目详情页
2. 点击「成员」标签
3. 点击「添加成员」按钮
4. 输入成员用户名或邮箱
5. 选择成员角色
6. 点击「确认」发送邀请

## 3. 功能指南

### 3.1 仪表盘

仪表盘是登录后的首页，展示：

- **我的项目**: 最近参与的项目列表
- **待办事项**: 分配给我的任务
- **活动动态**: 项目最新活动
- **统计数据**: 工作量统计图表

### 3.2 项目管理

#### 3.2.1 项目列表

项目列表页面展示所有可访问的项目：

- **搜索**: 按项目名称搜索
- **筛选**: 按状态、角色筛选
- **排序**: 按创建时间、名称排序

#### 3.2.2 项目详情

进入项目后可以查看：

- **概览**: 项目基本信息和统计
- **看板**: 可视化任务管理
- **用户故事**: 故事列表和详情
- **问题**: 问题跟踪列表
- **成员**: 项目成员管理
- **设置**: 项目配置

#### 3.2.3 项目设置

项目管理员可以在设置中：

- 修改项目名称和描述
- 更改项目可见性
- 启用/禁用功能模块
- 归档或删除项目

### 3.3 看板系统

#### 3.3.1 看板视图

看板以列的形式展示任务状态：

```
┌────────────┬────────────┬────────────┬────────────┐
│    待办    │   进行中   │   测试中   │    完成    │
├────────────┼────────────┼────────────┼────────────┤
│  ┌──────┐  │  ┌──────┐  │  ┌──────┐  │  ┌──────┐  │
│  │卡片1 │  │  │卡片3 │  │  │卡片5 │  │  │卡片7 │  │
│  └──────┘  │  └──────┘  │  └──────┘  │  └──────┘  │
│  ┌──────┐  │  ┌──────┐  │            │  ┌──────┐  │
│  │卡片2 │  │  │卡片4 │  │            │  │卡片8 │  │
│  └──────┘  │  └──────┘  │            │  └──────┘  │
└────────────┴────────────┴────────────┴────────────┘
```

#### 3.3.2 拖拽操作

1. 鼠标按住卡片
2. 拖动到目标列
3. 松开鼠标完成移动
4. 系统自动保存新状态

#### 3.3.3 卡片操作

点击卡片可以：

- 查看详细信息
- 编辑卡片内容
- 添加评论
- 更改指派人
- 删除卡片

#### 3.3.4 筛选功能

看板支持多种筛选条件：

- **指派人**: 筛选特定成员的任务
- **优先级**: 按优先级筛选
- **类型**: 用户故事/任务/问题
- **关键词**: 按标题搜索

### 3.4 用户故事

#### 3.4.1 创建用户故事

1. 进入项目的「用户故事」页面
2. 点击「新建故事」按钮
3. 填写故事信息：
   - **标题**: 故事名称
   - **描述**: 详细描述（支持 Markdown）
   - **故事点**: 工作量估算
   - **优先级**: 高/中/低
   - **指派给**: 负责人
4. 点击「创建」保存

#### 3.4.2 故事状态

用户故事支持以下状态流转：

```
新建 → 就绪 → 进行中 → 就绪测试 → 测试中 → 完成
                    ↓
                  阻塞
```

#### 3.4.3 故事详情

故事详情页展示：

- 基本信息（标题、描述、状态）
- 关联任务列表
- 评论和讨论
- 变更历史

### 3.5 问题跟踪

#### 3.5.1 创建问题

1. 进入项目的「问题」页面
2. 点击「新建问题」按钮
3. 填写问题信息：
   - **标题**: 问题概述
   - **描述**: 详细描述
   - **类型**: Bug/功能请求/改进
   - **严重程度**: 致命/严重/一般/轻微
   - **优先级**: 紧急/高/中/低
   - **指派给**: 处理人
4. 点击「创建」保存

#### 3.5.2 问题状态

问题支持以下状态：

| 状态 | 说明 |
|------|------|
| 新建 | 新提交的问题 |
| 确认 | 已确认的问题 |
| 进行中 | 正在处理 |
| 已解决 | 已修复待验证 |
| 已关闭 | 问题已解决 |

### 3.6 权限管理

#### 3.6.1 角色说明

| 角色 | 权限说明 |
|------|----------|
| **项目管理员** | 项目全部权限，包括成员管理 |
| **开发者** | 创建、编辑、查看项目内容 |
| **查看者** | 仅查看项目内容 |

#### 3.6.2 申请权限

1. 点击「权限」菜单
2. 选择目标项目
3. 选择申请的角色
4. 填写申请理由
5. 提交申请等待审批

#### 3.6.3 审批权限

项目管理员可以：

1. 查看待审批列表
2. 查看申请详情
3. 同意或拒绝申请
4. 添加审批备注

### 3.7 个人设置

#### 3.7.1 个人信息

在设置页面可以修改：

- 全名
- 邮箱
- 手机号
- 个人简介
- 头像

#### 3.7.2 安全设置

- 修改密码
- 查看登录历史
- 管理活跃会话

#### 3.7.3 偏好设置

- 语言设置
- 主题设置（亮色/暗色）
- 通知偏好

## 4. 常见问题

### 4.1 账号相关

**Q: 忘记密码怎么办？**

A: 在登录页面点击「忘记密码」，输入注册邮箱，系统会发送重置密码链接。

**Q: 如何修改用户名？**

A: 用户名创建后不可修改，如需更改请联系管理员。

**Q: 如何注销账号？**

A: 在设置页面底部找到「注销账号」选项，注销后数据将无法恢复。

### 4.2 项目相关

**Q: 项目归档后还能恢复吗？**

A: 可以。归档的项目可以在项目列表的「已归档」标签中找到，点击恢复即可。

**Q: 如何转让项目管理员权限？**

A: 在项目设置中，找到成员管理，将其他成员设为管理员，然后自己退出或降级。

**Q: 删除项目后数据能恢复吗？**

A: 项目删除后数据将永久删除，无法恢复，请谨慎操作。

### 4.3 看板相关

**Q: 卡片拖拽后状态没有更新？**

A: 请检查网络连接，刷新页面后重试。如果问题持续，请联系技术支持。

**Q: 如何自定义看板列？**

A: 项目管理员可以在项目设置中自定义看板状态列。

**Q: 看板加载很慢怎么办？**

A: 可能是数据量过大，建议使用筛选功能减少显示的卡片数量。

### 4.4 权限相关

**Q: 为什么看不到某些项目？**

A: 可能是没有该项目的访问权限，请联系项目管理员申请权限。

**Q: 权限申请多久会被审批？**

A: 这取决于项目管理员的处理速度，通常会在1-2个工作日内处理。

**Q: 如何查看自己的权限？**

A: 在权限页面可以查看自己在各项目中的角色和权限。

### 4.5 其他问题

**Q: 支持哪些浏览器？**

A: 推荐使用 Chrome、Firefox、Safari 或 Edge 的最新版本。

**Q: 数据安全吗？**

A: 系统采用加密传输和存储，定期备份数据，确保数据安全。

**Q: 如何获取技术支持？**

A: 请通过系统内的反馈功能或联系管理员获取帮助。
